常见问题

如何进行区域截图？

使用快捷键（默认 ⌘⇧1）或点击菜单栏图标选择「区域截图」，然后拖拽选择要截取的区域即可。

如何使用滚动截屏？

  1. 点击菜单栏图标选择「长截图」
  2. 拖拽选择要截取的区域
  3. 选择自动滚动或手动滚动模式
  4. 按 ↩ 键完成截图，按 ESC 取消

如何使用步进截图？

  1. 点击菜单栏图标选择「步进截图」
  2. 框选每个步骤的区域并添加说明
  3. 点击「完成」生成带步骤编号的教程图
  4. 点击「暂存」可暂时保存进度，稍后继续

如何使用 OCR 文字识别？

  1. 点击菜单栏图标选择「文字识别」
  2. 拖拽选择包含文字的区域
  3. 识别结果会自动复制到剪贴板

自动滚动截屏不滚动怎么办？

自动滚动需要辅助功能权限。请前往「系统设置 → 隐私与安全性 → 辅助功能」，确保 FocusCapture 已被授权，然后重新发起滚动截屏。

截图后没有出现编辑界面？

请检查「偏好设置 → 通用」中的「截图后操作」设置，确保已开启「显示编辑界面」选项。

如何购买 Pro 版本？

点击应用内的「升级到 Pro」按钮，通过 App Store 完成购买（一次性买断），即可永久解锁所有 Pro 功能。

如何恢复购买？

点击「升级到 Pro」页面中的「恢复购买」按钮即可。

系统要求

  • macOS 13.0 (Ventura) 或更高版本
  • Apple Silicon (M1/M2/M3/M4) 或 Intel 处理器
  • 建议 8GB 以上内存

FAQ

How do I take a region screenshot?

Use the shortcut (default ⌘⇧1) or click the menu bar icon and select "Region Screenshot", then drag to select the area.

How do I use scrolling capture?

  1. Click the menu bar icon and select "Scrolling Capture"
  2. Drag to select the area to capture
  3. Choose auto-scroll or manual-scroll mode
  4. Press ↩ to finish, ESC to cancel

How do I use Scribe (step-by-step capture)?

  1. Click the menu bar icon and select "Scribe"
  2. Select each step area and add a description
  3. Click "Finish" to generate a numbered tutorial image
  4. Click "Stash" to save progress and continue later

How do I use OCR text recognition?

  1. Click the menu bar icon and select "OCR"
  2. Drag to select the area containing text
  3. The recognized text is automatically copied to the clipboard

Auto-scroll doesn't work — what should I do?

Auto-scroll requires Accessibility permission. Go to System Settings → Privacy & Security → Accessibility and make sure FocusCapture is authorized, then try again.

The editor doesn't appear after taking a screenshot?

Check Preferences → General → "Action after screenshot" and make sure "Show editor" is enabled.

How do I purchase Pro?

Click the "Upgrade to Pro" button in the app and complete the purchase through the App Store (one-time purchase) to permanently unlock all Pro features.

How do I restore my purchase?

Click the "Restore Purchase" button on the "Upgrade to Pro" page.

よくある質問

範囲スクリーンショットの撮り方は？

ショートカット（デフォルト ⌘⇧1）を使用するか、メニューバーアイコンをクリックして「範囲スクリーンショット」を選択し、エリアをドラッグして選択します。

スクロールキャプチャの使い方は？

  1. メニューバーアイコンをクリックして「長いスクリーンショット」を選択
  2. キャプチャするエリアをドラッグして選択
  3. 自動スクロールまたは手動スクロールモードを選択
  4. ↩ キーで完了、ESC でキャンセル

ステップキャプチャ（Scribe）の使い方は？

  1. メニューバーアイコンをクリックして「ステップキャプチャ」を選択
  2. 各ステップのエリアを選択して説明を追加
  3. 「完了」をクリックして番号付きチュートリアル画像を生成
  4. 「一時保存」をクリックして後で続けることができます

OCRテキスト認識の使い方は？

  1. メニューバーアイコンをクリックして「テキスト認識」を選択
  2. テキストを含むエリアをドラッグして選択
  3. 認識されたテキストは自動的にクリップボードにコピーされます

自動スクロールが動作しない場合は？

自動スクロールにはアクセシビリティ権限が必要です。システム設定 → プライバシーとセキュリティ → アクセシビリティで FocusCapture が許可されていることを確認してから、再度お試しください。

スクリーンショット後にエディタが表示されない場合は？

環境設定 → 一般 →「スクリーンショット後の動作」を確認し、「エディタを表示」が有効になっているか確認してください。

Proバージョンの購入方法は？

アプリ内の「Proにアップグレード」ボタンをクリックし、App Storeで購入（買い切り）するとすべてのPro機能が永続的に使えます。

購入を復元するには？

「Proにアップグレード」ページの「購入を復元」ボタンをクリックしてください。

자주 묻는 질문

영역 스크린샷 찍는 방법은?

단축키(기본 ⌘⇧1)를 사용하거나 메뉴 막대 아이콘을 클릭하여 "영역 스크린샷"을 선택한 후 영역을 드래그하여 선택합니다.

스크롤 캡처 사용 방법은?

  1. 메뉴 막대 아이콘을 클릭하여 "긴 스크린샷" 선택
  2. 캡처할 영역을 드래그하여 선택
  3. 자동 스크롤 또는 수동 스크롤 모드 선택
  4. ↩ 키로 완료, ESC로 취소

단계별 캡처(Scribe) 사용 방법은?

  1. 메뉴 막대 아이콘을 클릭하여 "단계별 캡처" 선택
  2. 각 단계 영역을 선택하고 설명 추가
  3. "완료"를 클릭하여 번호가 매겨진 튜토리얼 이미지 생성
  4. "임시 저장"을 클릭하여 나중에 계속할 수 있습니다

OCR 텍스트 인식 사용 방법은?

  1. 메뉴 막대 아이콘을 클릭하여 "텍스트 인식" 선택
  2. 텍스트가 포함된 영역을 드래그하여 선택
  3. 인식된 텍스트가 자동으로 클립보드에 복사됩니다

자동 스크롤이 작동하지 않는 경우?

자동 스크롤은 손쉬운 사용 권한이 필요합니다. 시스템 설정 → 개인 정보 보호 및 보안 → 손쉬운 사용에서 FocusCapture가 허용되어 있는지 확인한 후 다시 시도하세요.

스크린샷 후 편집기가 표시되지 않는 경우?

환경설정 → 일반 → "스크린샷 후 동작"을 확인하고 "편집기 표시"가 활성화되어 있는지 확인하세요.

Pro 버전 구매 방법은?

앱 내 "Pro로 업그레이드" 버튼을 클릭하여 App Store에서 구매(일회성)하면 모든 Pro 기능을 영구적으로 사용할 수 있습니다.

구매 복원 방법은?

"Pro로 업그레이드" 페이지의 "구매 복원" 버튼을 클릭하세요.

시스템 요구사항

  • macOS 13.0 (Ventura) 이상
  • Apple Silicon (M1/M2/M3/M4) 또는 Intel 프로세서
  • RAM 8GB 이상 권장
