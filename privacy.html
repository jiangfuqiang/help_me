BatchMark 隐私政策

生效日期：2026-03-27 | 最后更新：2026-03-27

简介

BatchMark 尊重并保护所有用户的隐私权。本政策说明我们如何处理您的信息。

数据收集

完全不收集任何数据

BatchMark 是一款完全本地化的图片处理工具，不会收集：
  • 您的图片文件
  • 个人身份信息
  • 设备标识符
  • 位置信息
  • 使用行为数据

数据处理

  • 所有图片处理均在您的设备本地完成
  • 处理后的图片保存在您选择的本地目录
  • 不会上传任何文件到服务器
  • 不会通过网络传输任何数据

权限说明

文件访问权限

用于读取您选择的图片文件，保存处理后的图片。仅限您明确选择的文件和文件夹。

下载文件夹访问权限

用于保存处理后的图片到下载文件夹。可选权限，您可以选择其他位置保存文件。

App 内购买

购买通过 Apple App Store 处理，我们不会获取您的支付信息。购买记录由 Apple 管理。

第三方服务

BatchMark 不包含任何第三方分析工具、广告 SDK 或社交媒体 SDK。仅使用 Apple 提供的系统框架：

  • Vision Framework（人脸检测）
  • CoreImage（图片处理）
  • StoreKit（App 内购买）

联系我们

如有隐私相关问题，请联系：fc_help@163.com

BatchMark Privacy Policy

Effective Date: 2026-03-27 | Last Updated: 2026-03-27

Introduction

BatchMark respects and protects the privacy of all users. This policy explains how we handle your information.

Data Collection

We do not collect any data

BatchMark is a fully local image processing tool. We do not collect:
  • Your image files
  • Personal identification information
  • Device identifiers
  • Location information
  • Usage behavior data

Data Processing

  • All image processing is done locally on your device
  • Processed images are saved to your chosen local directory
  • No files are uploaded to any server
  • No data is transmitted over the network

Permissions

File Access

Used to read your selected image files and save processed images. Limited to files and folders you explicitly select.

Downloads Folder Access

Used to save processed images to the Downloads folder. Optional permission; you can choose other save locations.

In-App Purchases

Purchases are processed through Apple App Store. We do not access your payment information. Purchase records are managed by Apple.

Third-Party Services

BatchMark does not include any third-party analytics, advertising SDKs, or social media SDKs. We only use Apple system frameworks:

  • Vision Framework (face detection)
  • CoreImage (image processing)
  • StoreKit (in-app purchases)

Contact Us

For privacy-related questions, contact: fc_help@163.com

BatchMark プライバシーポリシー

発効日：2026-03-27 | 最終更新：2026-03-27

はじめに

BatchMarkはすべてのユーザーのプライバシーを尊重し保護します。このポリシーでは、お客様の情報の取り扱いについて説明します。

データ収集

データを一切収集しません

BatchMarkは完全にローカルな画像処理ツールです。以下を収集しません：
  • お客様の画像ファイル
  • 個人を特定できる情報
  • デバイス識別子
  • 位置情報
  • 使用行動データ

データ処理

  • すべての画像処理はお客様のデバイス上でローカルに行われます
  • 処理された画像は選択したローカルディレクトリに保存されます
  • ファイルはサーバーにアップロードされません
  • ネットワークを介してデータが送信されることはありません

権限について

ファイルアクセス

選択した画像ファイルの読み取りと処理後の画像の保存に使用されます。明示的に選択したファイルとフォルダに限られます。

ダウンロードフォルダへのアクセス

処理後の画像をダウンロードフォルダに保存するために使用されます。オプションの権限です。他の保存場所を選択できます。

アプリ内購入

購入はApple App Storeを通じて処理されます。お客様の支払い情報にアクセスすることはありません。購入記録はAppleが管理します。

サードパーティサービス

BatchMarkにはサードパーティの分析ツール、広告SDK、ソーシャルメディアSDKは含まれていません。Appleのシステムフレームワークのみを使用します：

  • Vision Framework（顔検出）
  • CoreImage（画像処理）
  • StoreKit（アプリ内購入）

お問い合わせ

プライバシー関連のご質問はこちらまで：fc_help@163.com

BatchMark 개인정보 처리방침

발효일: 2026-03-27 | 최종 업데이트: 2026-03-27

소개

BatchMark는 모든 사용자의 개인정보를 존중하고 보호합니다. 이 정책은 귀하의 정보를 어떻게 처리하는지 설명합니다.

데이터 수집

어떠한 데이터도 수집하지 않습니다

BatchMark는 완전히 로컬인 이미지 처리 도구입니다. 다음을 수집하지 않습니다:
  • 귀하의 이미지 파일
  • 개인 식별 정보
  • 기기 식별자
  • 위치 정보
  • 사용 행동 데이터

데이터 처리

  • 모든 이미지 처리는 귀하의 기기에서 로컬로 수행됩니다
  • 처리된 이미지는 선택한 로컬 디렉토리에 저장됩니다
  • 파일이 서버에 업로드되지 않습니다
  • 네트워크를 통해 데이터가 전송되지 않습니다

권한 안내

파일 접근

선택한 이미지 파일을 읽고 처리된 이미지를 저장하는 데 사용됩니다. 명시적으로 선택한 파일과 폴더로만 제한됩니다.

다운로드 폴더 접근

처리된 이미지를 다운로드 폴더에 저장하는 데 사용됩니다. 선택적 권한이며 다른 저장 위치를 선택할 수 있습니다.

인앱 구매

구매는 Apple App Store를 통해 처리됩니다. 귀하의 결제 정보에 접근하지 않습니다. 구매 기록은 Apple이 관리합니다.

제3자 서비스

BatchMark에는 제3자 분석 도구, 광고 SDK 또는 소셜 미디어 SDK가 포함되어 있지 않습니다. Apple 시스템 프레임워크만 사용합니다:

  • Vision Framework (얼굴 감지)
  • CoreImage (이미지 처리)
  • StoreKit (인앱 구매)
