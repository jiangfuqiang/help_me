常见问题

如何使用批量水印？

  1. 选择"水印"标签
  2. 拖拽或选择要处理的图片
  3. 配置水印文字、位置、透明度
  4. 选择输出目录，点击"开始处理"

如何使用批量打码？

  1. 选择"打码"标签
  2. 拖拽或选择要处理的图片
  3. 勾选需要检测的类型（人脸、手机号等）
  4. 调整打码样式，点击"开始处理"

支持哪些图片格式？

JPG/JPEG、PNG、HEIC、TIFF、BMP

如何购买 Pro 版本？

应用内点击 Pro 升级按钮，通过 App Store 一次性购买（¥35），永久使用所有 Pro 功能。

如何恢复购买？

点击"恢复购买"按钮即可。

系统要求

  • Apple Silicon (M1/M2/M3/M4) 或 Intel 处理器
  • 建议 8GB 以上内存

FAQ

How to use batch watermark?

  1. Select the "Watermark" tab
  2. Drag or select images to process
  3. Configure watermark text, position, and opacity
  4. Select output folder and click "Start Processing"

How to use batch masking?

  1. Select the "Mask" tab
  2. Drag or select images to process
  3. Select detection types (face, phone number, etc.)
  4. Adjust mask style and click "Start Processing"

Supported image formats?

JPG/JPEG, PNG, HEIC, TIFF, BMP

How to purchase Pro version?

Click the Pro upgrade button in the app and purchase through App Store ($4.99 one-time) to unlock all Pro features permanently.

How to restore purchase?

Click the "Restore Purchase" button.

System Requirements

  • Apple Silicon (M1/M2/M3/M4) or Intel processor
  • 8GB RAM recommended

よくある質問

バッチウォーターマークの使い方

  1. 「ウォーターマーク」タブを選択
  2. 画像をドラッグまたは選択
  3. テキスト、位置、透明度を設定
  4. 出力フォルダを選択し「処理開始」をクリック

バッチモザイクの使い方

  1. 「モザイク」タブを選択
  2. 画像をドラッグまたは選択
  3. 検出タイプを選択（顔、電話番号など）
  4. モザイクスタイルを調整し「処理開始」をクリック

対応画像形式

JPG/JPEG、PNG、HEIC、TIFF、BMP

Pro版の購入方法

アプリ内のProアップグレードボタンからApp Storeで購入（¥600・ワンタイム）し、すべてのPro機能を永久にご利用いただけます。

購入の復元方法
